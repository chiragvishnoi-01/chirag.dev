# Chirag Vishnoi | Premium Developer Portfolio

![Portfolio Hero](public/hero.png)

## ✨ Overview
An immersive, ultra-premium digital experience that serves as a personal portfolio. Built with a focus on cinematic storytelling, smooth interactions, and cutting-edge 3D graphics. This project pushes the boundaries of standard web portfolios by integrating physics-based animations, procedural shaders, and a custom smooth-scrolling engine.

## 🚀 Key Features
- **Cinematic 3D Scene**: Powered by Three.js and React Three Fiber, featuring custom GLSL shaders and interactive particles.
- **Micro-Interactions**: Smooth GSAP and Framer Motion transitions that breathe life into every scroll and click.
- **Modern Smooth Scroll**: Integration of Lenis for a butter-smooth, high-end scrolling feel.
- **Glassmorphism UI**: A sleek, translucent design language that complements the 3D background.
- **AI Integration**: Experimental features utilizing Gemini API for dynamic content interactions.
- **Fully Responsive**: Optimized for high-end desktop experiences while maintaining performance on mobile devices.

## 📦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [NPM](https://www.npmjs.com/)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/chiragvishnoi-01/chirag.dev.git
   cd chirag.dev
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root directory and add your keys:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run development server:**
   ```bash
   npm run dev
   ```

## 🏗️ Deployment
To create a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` folder, ready to be hosted on platforms like Vercel, Netlify, or GitHub Pages.
